# Projeto Glass HTML5

Site multipágina desenvolvido com foco em estruturação semântica utilizando HTML5 e estilização com CSS3.


---

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

---

## 📌 Objetivo
Praticar a construção de um site completo com múltiplas páginas, aplicando boas práticas de semântica HTML e organização de layout.

---

## 📚 O que eu aprendi
- Estruturação semântica com HTML5
- Organização de navegação entre páginas
- Estilização com CSS3
- Estrutura de layout tradicional para websites
